雨中小结

2019-03-20 阅读次数 0

近来可以说是诸事不顺了，偏偏又在约人见面的时候赶上下阵雨，感觉好生凄凉……

从学期开始，准确说是上学期末实验就做的不是很顺，值得庆幸的是昨天改变方案以后代码结构变化不大，所以2d版本的训练应该这两日就可以开始跑起来了，算是不幸中的万幸吧，希望这版跑起来以后上至少3个点，这样也好对我提了好几版的特征有个交代，否则过年时候跑的代码炼的丹全都白费了。对目前的失败总结一下，大概3d cnn的特征确实太容易被破坏了，根据实验结果猜测很难通过自己加入的特征或是mask学到什么，因为tune时候可视化部分卷积层以后并没有什么显著的融合效果，大概率成为了冗余信息，甚至可能都没有背景信息在3d卷积中的作用大，其实早在2月就应该尝试一下2d版本的实验，可惜自己一直没搞懂原理，浪费的时间可能近期要好好弥补一下了，写代码搬砖容易，做一些真正有意义的改动还是难。从做pose看人家一个组把hourglass或是cpn玩的五花八门，自己就是写不出那样的文章，其实在baseline上的改进并不复杂，但是formulation和故事就很难想出像样的东西，不得不说对于想在学术界深耕的人来说，组里的氛围导向和老师指导确实很重要，论文发起来确实有套路，而且学的多好对研究生来说并不意味着能在科研上做到多好，想想自己确实在科研素质上有所欠缺，当然我觉得目标是拿学位的人都不会对科研素质有什么想法，以前我觉得coding+对研究有一定了解是够的，现在发现coding能力比普通人强一点不够强，基本由你编程起步时间决定了，而算法功力更是对普通人不友好，理解和了解模型和表面上这些皮毛的视觉机制并不代表足够的创造基础，可以说这个研究生生涯过的相当失败了，但是问题确实还是出在自身，没有找准定位，属实应该反思。

反思完近期的工作，该反思持续好久的迷茫期了。不幸的人各有不同，但是我就总是特别拔群。我一直有个疑惑，为什么出事的总是我，别人怎么就总是过得非常平稳。一直以来我都觉得是我自己的问题，肯定是我哪做的有问题，才导致厄运总是降到我脑袋上，但是现在我大概明白了一个问题，大概还是我遇人不淑，然后导致我心理素质越来越差，思维方式越来越畸形。教育很重要，我觉得一直以来我接受的亲朋好友以外的说教都是只有教学，没有教育。人生思想最关键的时段我被塞满了语数外物化生导致没空思考这些问题，然后有了个不高不低的起点，之后就一直按照这个固定的斜率发展下去，但是从来没人告诉我怎么摆脱现状，当然了确实没人有义务告诉我这一点，但是目前的人生天花板确实受制于我前期人生的有限资源和选择余地。我从小受到的教育就从努力考个好大学一下转型为找准自己定位，一方面是畸形的教学让我没空琢磨，二是我确实也没条件自己思考这个问题。但是最近遇到形形色色的人，深感自己确实之前遇人不淑，如果能和有些人生导师早点打个交道，也不至于目前这种处境，只能说信息不对等造就了很多悲剧，明天还要有艰苦卓绝的战斗要打，先琢磨到这吧……可别再下雨了，下雨天心情太差了。
